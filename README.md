### Overview
**GEEK HUD** is a premium, single-screen data field designed for Garmin Edge cycling computers (540 / 840 / 1040 / 1050). Inspired by sci-fi heads-up displays and military interfaces, it provides all your crucial cycling metrics in one highly legible, high-contrast dashboard. 

### Core Features
* [cite_start]**Dynamic Speed Gauge:** A cyan arc that fills based on your current speed[cite: 42]. [cite_start]It turns warning red if you exceed the maximum speed limit[cite: 42].
* [cite_start]**Real-time Power Zone:** The polygon below the gauge dynamically changes shape and color (7 Zones) based on your 3-second average power and FTP[cite: 17, 18, 64, 68].
* [cite_start]**Comprehensive Metrics:** * Speed (km/h) & Average Speed [cite: 70, 71]
    * [cite_start]3s Power [cite: 72]
    * [cite_start]Heart Rate [cite: 73, 74]
    * [cite_start]Grade (%) [cite: 75, 76]
    * [cite_start]Distance (km) & Total Ascent (m) [cite: 77, 78]
    * [cite_start]Cadence [cite: 79]
    * [cite_start]Ride Time [cite: 80]

### âš ï¸ IMPORTANT: Layout Setup (1-Field Only)
To maintain the integrity of the complex vector graphics and HUD design, **this data field ONLY supports the 1-Field (Full Screen) layout.** **How to set it up:**
1. On your Edge device, go to **Activity Profiles** > Select your profile (e.g., Road).
2. Go to **Data Screens** > Add new or edit an existing screen.
3. Set **Layout and Data Fields** to **1 Field** (the entire screen).
4. Select `Connect IQ` > `GEEK HUD`.
*(Note: If placed in a split-screen layout, the UI will display a warning to change the layout.)*
