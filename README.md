## English

### Overview
**Cyber Ride** is a premium, single-screen data field designed for Garmin Edge cycling computers (540 / 840 / 1040 / 1050). Inspired by sci-fi heads-up displays and military interfaces, it provides all your crucial cycling metrics in one highly legible, high-contrast dashboard. 

### Core Features
* [cite_start]**Dynamic Speed Gauge:** A cyan arc that fills based on your current speed[cite: 42]. [cite_start]It turns warning red if you exceed the maximum speed limit[cite: 42].
* [cite_start]**Real-time Power Zone:** The polygon below the gauge dynamically changes shape and color (7 Zones) based on your 3-second average power and FTP[cite: 17, 18, 64, 68].
* [cite_start]**Comprehensive Metrics:** * Speed (km/h) & Average Speed [cite: 70, 71]
    * [cite_start]3s Power [cite: 72]
    * [cite_start]Heart Rate [cite: 73, 74]
    * [cite_start]Grade (%) [cite: 75, 76]
    * [cite_start]Distance (km) & Total Ascent (m) [cite: 77, 78]
    * [cite_start]Cadence [cite: 79]
    * [cite_start]Ride Time [cite: 80]

### ⚠️ IMPORTANT: Layout Setup (1-Field Only)
To maintain the integrity of the complex vector graphics and HUD design, **this data field ONLY supports the 1-Field (Full Screen) layout.** **How to set it up:**
1. On your Edge device, go to **Activity Profiles** > Select your profile (e.g., Road).
2. Go to **Data Screens** > Add new or edit an existing screen.
3. Set **Layout and Data Fields** to **1 Field** (the entire screen).
4. Select `Connect IQ` > `GEEK HUD`.
*(Note: If placed in a split-screen layout, the UI will display a warning to change the layout.)*

## 中文说明

### 简介
**Cyber Ride** 是一款专为 Garmin Edge 系列（540 / 840 / 1040 / 1050）打造的高级单屏数据字段。采用硬核机甲与军事风格的 HUD（平视显示器）设计，在提供极高对比度的同时，将所有核心骑行数据集成在一个屏幕中。

### 核心功能
* [cite_start]**动态速度仪表盘：** 青色实心圆弧会随当前速度动态填充 [cite: 42][cite_start]，当突破极限速度时会变为红色警示 [cite: 42]，配合背景的射线切割设计，极具速度感。
* [cite_start]**实时功率区间：** 仪表盘下方的多边形模块会根据你的 3 秒平均功率与 FTP，实时切换 7 种区间颜色与形态 [cite: 17, 18, 64, 68]。
* [cite_start]**全维数据展示：** * 当前速度 (km/h) 与 平均速度 [cite: 70, 71]
    * [cite_start]3秒平均功率 [cite: 72]
    * [cite_start]实时心率 [cite: 73, 74]
    * [cite_start]当前坡度 (%) [cite: 75, 76]
    * [cite_start]骑行距离 (km) 与 总爬升 (m) [cite: 77, 78]
    * [cite_start]踏频 [cite: 79]
    * [cite_start]骑行时间 [cite: 80]

### ⚠️ 关键设置：仅支持全屏布局 (1-Field)
为了保证复杂的矢量图形和 HUD 界面不被拉伸变形，**本数据字段严格限制为“单栏（全屏）”布局。**

**设置步骤：**
1. 在 Edge 码表上，进入 **活动模式** > 选择你的常用模式（如：公路骑行）。
2. 进入 **数据页面** > 新增或编辑现有页面。
3. 将 **布局和数据字段** 设置为 **1 栏**（即占据整个屏幕）。
4. 选择 `Connect IQ` > 找到并选中 `GEEK HUD`。
*（注：如果放置在多栏分屏布局中，界面将弹出警告，提示您更改布局。）*
